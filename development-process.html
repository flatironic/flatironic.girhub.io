DEVELOPMENT PROCESS:
Visual Concepts | Code Concepts | Engineering | Testing

Visual concepts are usually built using a combination of Photoshop, code generated screen shots and inVision. Rounds of collaboration and refinement allow future apps to evolve into a collective vision.

Code concepts take proposed building blocks of an application and "prove the concept" of how they get applied.

Engineering is a process of defining app architecture and writing code to match visual concepts and predetermined requirements.

Testing is performed throughout the engineering process. Defining test cases as they get built to eventually build a list of tests that can be applied to a regression testing plan. Both automated tests and actual real world people that can discover bugs and offer insights.
